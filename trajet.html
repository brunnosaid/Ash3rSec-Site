Complete Beginner

Quais os desafios?

O "Beginner Path" visa proporcionar uma ampla introdução às diferentes áreas da Segurança de Computadores. Este path abordará as seguintes áreas:

  • Basic Linux - Familiarize-se com a linha de comando do Linux. Web Application Security - Aprenda conceitos de segurança de aplicações web através do OWASP Top 10.
  • Network Security - Utilize ferramentas essenciais como o NMAP para enumerar a infraestrutura.
  • Scripting Challenges - Utilize Python e Bash para realizar diferentes tarefas.
  • Privilege Escalation

Uma vez que você complete o caminho para iniciantes, você deverá ter adquirido o conhecimento fundamental em cada área específica e usar esses conceitos básicos para construir sua compreensão sobre tópicos mais complexos dentro da área.

Web Fundamentals

Quais os desafios?

O objetivo deste caminho é ensinar você a atacar aplicações web. Para atacar e explorar aplicações web com sucesso, você precisa entender como elas funcionam. Após concluir este path, você será capaz de:

  • Compreender como as aplicações web funcionam
  • Utilizar ferramentas padrão da indústria ao atacar aplicações web
  • Explicar e explorar vulnerabilidades web comuns
  • Aplicar esse conhecimento a outros alvos
